# Обновление кнопок админ панели в стиле Telegram

## Обзор

Кнопки функций админ панели обновлены до дизайна, аналогичного настройкам группы в Telegram. Вместо карточек с градиентами использован минималистичный список с иконками и разделителями.

## Дизайн в стиле Telegram

### ✅ **Основные характеристики:**

1. **Списковый layout** - вертикальный список вместо сетки
2. **Минималистичный дизайн** - без градиентов и теней
3. **Иконки в контейнерах** - квадратные иконки с фоном
4. **Разделители** - тонкие линии между элементами
5. **Стрелка навигации** - символ "›" справа
6. **Hover эффекты** - плавное изменение фона

## Структура элементов

### HTML структура:
```html
<div class="function-item">
    <div class="function-content">
        <div class="function-icon">👥</div>
        <div class="function-text">
            <div class="function-name">Просмотр пользователей</div>
            <div class="function-description">Просмотр списка пользователей чата</div>
        </div>
        <div class="function-arrow">›</div>
    </div>
    <div class="function-divider"></div>
</div>
```

### CSS стили:

#### Основной контейнер:
```css
.functions-grid {
    display: flex;
    flex-direction: column;
    gap: 0;
    width: 100%;
    max-width: 1200px;
}
```

#### Элемент списка:
```css
.function-item {
    background: rgba(255, 255, 255, 0.05);
    border: none;
    border-radius: 0;
    cursor: pointer;
    transition: background-color 0.2s ease;
}

.function-item:hover {
    background: rgba(255, 255, 255, 0.08);
}

.function-item:active {
    background: rgba(255, 255, 255, 0.12);
}
```

#### Иконка:
```css
.function-icon {
    width: 40px;
    height: 40px;
    border-radius: 10px;
    background: rgba(179, 136, 255, 0.15);
    display: flex;
    align-items: center;
    justify-content: center;
    font-size: 20px;
    color: #b388ff;
    flex-shrink: 0;
}
```

#### Текст:
```css
.function-name {
    font-size: 16px;
    font-weight: 500;
    color: #ffffff;
    margin-bottom: 2px;
}

.function-description {
    font-size: 14px;
    color: rgba(255, 255, 255, 0.6);
    line-height: 1.3;
}
```

#### Разделитель:
```css
.function-divider {
    height: 1px;
    background: rgba(255, 255, 255, 0.08);
    margin: 0 20px;
}
```
