
Нужно добавить в админ-панель чата новую функцию «Администраторы».

1. Раздел в админ-панели

В главном меню админ-панели должен появиться пункт «Администраторы».
При нажатии он открывает страницу управления администраторами.

2. Добавление администратора

На странице должна быть возможность:
 • добавить нового администратора по ID / username / email (в зависимости от системы пользователей)
 • открыть панель настройки прав, аналогичную системе выдачи прав администратора в группах в Telegram

3. Настройка прав

Права должны задаваться через тумблеры (вкл/выкл).

Примеры прав:
 • доступ к админ-панели
 • управление пользователями
 • модерация сообщений
 • управление администраторами
 • просмотр аналитики
 • управление настройками

Каждое право — отдельный toggle switch.

4. Сохранение на backend

При добавлении администратора:
 • данные записываются в базу данных
 • сохраняются:
 • user_id
 • role = admin
 • список прав (permissions[])

Пример структуры:

admins
- id
- user_id
- permissions (json / array)
- created_at

или

permissions
- admin_id
- permission_name
- enabled

5. Проверка прав

Когда администратор заходит в админ-панель:
 1. backend проверяет его user_id
 2. получает список прав из базы
 3. frontend отображает только те разделы и функции, на которые у него есть разрешение

Например:
 • если нет права manage_admins → скрыть раздел Администраторы
 • если нет analytics → скрыть аналитику

6. Список администраторов

На странице «Администраторы» также должен быть:
 • список всех администраторов
 • их права
 • кнопки:
 • редактировать права
 • удалить администратора
