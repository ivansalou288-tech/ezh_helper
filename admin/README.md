# Админ панель Telegram бота

## Описание

Админ панель представляет собой Telegram Mini App, которая позволяет пользователям с административными правами управлять чатами через веб-интерфейс. Доступные функции зависят от прав пользователя в базе данных.

## Структура проекта

```
admin/
├── client/
│   ├── index.html          # Главная страница со списком админских чатов
│   └── chat_admin.html     # Админ панель конкретного чата
├── server/
│   ├── api.py             # API сервера
│   └── run.py             # Скрипт для запуска сервера
└── README.md              # Этот файл
```

## Права доступа

Система поддерживает следующие права:

- `can_see_users` - Просмотр пользователей чата
- `can_do_admin` - Административные функции (управление пользователями, предупреждения)
- `can_recom` - Управление рекомендациями
- `can_links` - Управление ссылками
- `can_dk` - Специальные DK функции

## Запуск

### 1. Запуск основного API сервера

```bash
cd d:\ezh_helper
python api.py
```

Основной сервер будет доступен на порту 3000.

### 2. Запуск сервера админ панели (опционально)

```bash
cd d:\ezh_helper\admin\server
python run.py
```

Сервер админ панели будет доступен на порту 3001.

## Использование

### 1. Настройка Telegram Bot

1. Создайте бота через @BotFather
2. Получите токен бота
3. Настройте домен для Mini App (требуется HTTPS)

### 2. Доступ к админ панели

1. Откройте главную страницу: `https://your-domain.com/admin/client/index.html`
2. Авторизуйтесь через Telegram Web App
3. Выберите чат из списка доступных

### 3. Функции

- **Главная страница**: Показывает список чатов, где у пользователя есть админские права
- **Админ панель чата**: Показывает доступные функции в зависимости от прав
- **Просмотр пользователей**: Отображает список пользователей с их рангами и активностью

## API эндпоинты

### Основной сервер (порт 3000)

- `GET /user-admin-chats/{user_id}` - Получение админских чатов пользователя
- `GET /chat-avatar/{chat_id}` - Получение аватара чата
- `GET /chat-admin-panel/{chat_id}/{user_id}` - Получение данных админ панели чата
- `GET /chat-users/{chat_id}` - Получение пользователей чата

## База данных

### admin.db

Таблица `admins`:
- `user_id` - ID пользователя Telegram
- `chat_id` - ID чата
- `chat_name` - Название чата
- `can_see_users` - Право на просмотр пользователей
- `can_do_admin` - Право на административные функции
- `can_recom` - Право на управление рекомендациями
- `can_links` - Право на управление ссылками
- `can_dk` - Право на DK функции

### Базы данных чатов

Каждый чат имеет свою базу данных в папке `databases/` с именем `{chat_id}.db`.

Таблица `users`:
- `tg_id` - ID пользователя Telegram
- `nik` - Ник пользователя
- `rang` - Ранг (0-6)
- `date_vhod` - Дата входа в чат
- `last_date` - Последняя активность

## Разработка

### Добавление новых функций

1. Добавьте право в базу данных `admin.db`
2. Обновите функцию `get_chat_admin_panel` в `api.py`
3. Добавьте обработчик в `chat_admin.html`
4. Обновите стили при необходимости

### Кастомизация

- Измените цвета и стили в CSS секциях HTML файлов
- Обновите иконки функций в функции `getFunctionIcon`
- Измените тексты и сообщения в JavaScript коде

## Безопасность

- Все запросы проверяют права пользователя
- Доступ к функциям ограничен правами из БД
- Используется HTTPS для Telegram Web App
- Валидация всех входных данных

## Требования

- Python 3.8+
- FastAPI
- Uvicorn
- SQLite3
- Telegram Bot API токен
