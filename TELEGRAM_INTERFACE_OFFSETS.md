# Обновление отступов для интерфейса Telegram

## Обзор

Все страницы админ панели обновлены для обеспечения минимального отступа 95px сверху, чтобы избежать перекрытия элементов интерфейсом Telegram Mini App.

## Внесенные изменения

### 1. Основные отступы (desktop)

**Во всех файлах обновлено:**
- `.user-profile` - `top: 95px` (было 20px)
- `.main-content` - `margin-top: 95px` (было 80px)
- `min-height` - `calc(100vh - 135px)` (было 120px)

### 2. Мобильные отступы (@media max-width: 768px)

**Во всех файлах обновлено:**
- `.main-content` - `margin-top: 115px` (было 120px)
- `.user-profile` - `top: 95px` (было 90px)

### 3. Обновленные файлы:

✅ **index.html** - Главная страница
✅ **chat_admin.html** - Админ панель чата  
✅ **see_users.html** - Просмотр пользователей
✅ **admin_functions.html** - Административные функции
✅ **warn_management.html** - Управление предупреждениями
✅ **recommendations.html** - Рекомендации
✅ **links.html** - Ссылки
✅ **dk_functions.html** - DK функции

## CSS изменения

### Desktop стили:
```css
.user-profile {
    position: fixed;
    top: 95px;        /* было 20px */
    right: 20px;
    /* ... */
}

.main-content {
    padding: 20px;
    margin-top: 95px;  /* было 80px */
    min-height: calc(100vh - 135px); /* было 120px */
    /* ... */
}
```

### Mobile стили:
```css
@media (max-width: 768px) {
    .user-profile {
        top: 95px;        /* было 90px */
        /* ... */
    }
    
    .main-content {
        margin-top: 115px; /* было 120px */
        /* ... */
    }
}
```

## Причина изменений

Интерфейс Telegram Mini App может занимать до 95px высоты в верхней части экрана, включая:
- Заголовок приложения
- Кнопку "Назад" 
- Системные элементы интерфейса

Чтобы избежать перекрытия контента, все элементы сдвинуты ниже на безопасное расстояние.

## Результат

- ✅ Все элементы теперь находятся минимум на 95px ниже верха экрана
- ✅ Профиль пользователя позиционирован с учетом интерфейса Telegram
- ✅ Контент не перекрывается системными элементами
- ✅ Адаптивные отступы для мобильных устройств
- ✅ Сохранена единая стилистика и функциональность

Если потребуется дополнительная корректировка отступов:

- Mobile: изменить `margin-top: 115px` в медиа-запросах
- Учесть высоту конкретных элементов интерфейса Telegram
